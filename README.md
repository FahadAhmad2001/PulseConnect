# PulseConnect

### PulseConnect is a fully open-source electronic healthcare management system

Currently a work-in-progress

### Currently developing features

- Managing patient electronic medical records digitally
- Web-based management for admin and web client for doctors, nursing, ancillary and technical staff, and patients
- PC, iOS, android applications
- Templates for accurate and detailed progress note generation
- Advanced inter-specialty consultation system
- Integrated laboratory and radiological management
- Integration with clinical devices including cardiorespiratory monitors, ventilators, infusion pumps, anesthesia machines
- Integrated clinical scoring systems, e.g. CURB-65, Well's score, T4 score, CHA2DS2-VASc score, PRAM score, among others
- Integrated alert system for when lab / radiology results are out
- Integrated rapid response system for alerting, co-ordinating, recording, and reviewing results of rapid responses in a hospital (including code blues)
